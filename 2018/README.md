### Getting up and running

Use latest version of Node (v10.6.0)
Preferably use Node Version Manager (https://github.com/creationix/nvm)

Install dev dependencies:
```
npm install --dev
```

To build:
```
npm run build
```

To run:
```
npm run start
```
